33333PGP provides extensive keyboard shortcuts to help you navigate efficiently through the platform. This guide covers all available keyboard shortcuts organized by context and function.

## Basic Navigation

### Table Navigation

-   **↑ (Up Arrow)**: Move to previous row in a table
-   **↓ (Down Arrow)**: Move to next row in a table
-   **Option + ↑**: Navigate to previous page
-   **Option + ↓**: Navigate to next page
-   **Fn + ↑**: Scroll up within current section
-   **Fn + ↓**: Scroll down within current section

### Drawer Navigation

-   **→ (Right Arrow)**: Navigate to next tab in drawer
-   **← (Left Arrow)**: Navigate to previous tab in drawer
-   **Right Arrow**: Open drawer (when in Preseed view)
-   **Left Arrow**: Close drawer (when in Preseed view)
-   **ESC**: Close drawer (in all views except Preseed)
-   **Option + ←**: Switch to table navigation while staying in the current tab

## Context-Specific Actions

### Vulnerabilities Page (Ctrl + Key)

-   **Opt + T**: Mark vulnerability as Pending Triage
-   **Opt + O**: Mark vulnerability as Opened
-   **Opt + C**: Mark vulnerability as Closed
-   **Opt + A**: Mark vulnerability as Accepted
-   **Opt + R**: Mark vulnerability as Remediated

### Assets Page (Ctrl + Key)

-   **Opt + F**: Freeze selected asset(s)
-   **Opt + D**: Delete selected asset(s)

### Seeds Page (Ctrl + Key)

-   **Opt + A**: Approve selected seed(s)
-   **Opt + R**: Reject selected seed(s)
-   **Opt + D**: Delete selected seed(s)
-   **Opt + F**: Freeze selected seed(s)

## Tips for Keyboard Navigation

1.  **Table Selection**
    -   Use arrow keys to move between rows
    -   Selected row will automatically center in view
    -   First press of down arrow selects first row if nothing is selected
2.  **Drawer Interaction**
    -   Arrow keys switch between tabs when drawer is open
    -   Right/Left arrows can toggle drawer open/closed in certain views
3.  **Page Scrolling**
    -   Use Fn + Arrow keys for scrolling within sections without changing selection
    -   Option + Arrow keys help navigate between major sections or pages
4.  **Bulk Actions**
    -   Most Ctrl + Key shortcuts can be applied to multiple selected items
    -   Select multiple items before using shortcuts for bulk operations
5.  **Status Changes**
    -   Ctrl + Key combinations provide quick access to common status changes
    -   Different status options are available depending on the current page context

Remember that keyboard shortcuts are designed to complement mouse navigation, providing a faster alternative for common operations. As you become familiar with these shortcuts, you'll find they significantly speed up your workflow in PGP.
