The PGP **Metrics** dashboard serves as your central command center for monitoring and managing your security posture. The interface is thoughtfully organized into three main sections—Vulnerabilities, Assets, and Remediation—each offering unique insights through customizable widgets that help you understand and improve your security stance.

## Default Layout

### Vulnerabilities Section

The Vulnerabilities section provides a comprehensive view of your security landscape through several key widgets. The Noise Reduction widget helps you distinguish between active threats and resolved issues by displaying both open and remediated vulnerabilities side by side.

## Dashboard Customization

The dashboard is designed to adapt to your specific needs through intuitive customization options. You'll find an Add button in the top right corner that allows you to incorporate additional widgets into any section.

![](https://69448200be4d4ffad9065e4d.featurebase-attachments.com/c/article/69580d0aa21178a432dac4a7/019b7ff2-ee9e-7f1e-a40d-210777c7f3a0/b64u-MDE5YjdmZjItZWRlZS03OTQzLTg5YWMtODM4NzEwYjgwZjY2.png?X-Amz-Expires=3600&X-Amz-Date=20260318T050000Z&X-Amz-Algorithm=AWS4-HMAC-SHA256&X-Amz-Credential=DO801TYC4FCVNNEKURKM%2F20260318%2Ffra1%2Fs3%2Faws4_request&X-Amz-SignedHeaders=host&X-Amz-Signature=f69f18767190297db8dbe33e1abb986502c05299533d795efd3c2a78ae67251d)

When you need to refine the layout, the Edit button enables you to resize and reposition widgets through simple drag-and-drop interactions. During editing, you have the flexibility to adjust the dashboard until it perfectly suits your workflow. Once you're satisfied with your changes, simply click Done to lock them in place.

For those times when you need to start fresh, the platform includes a Reset to Default Layout option within the editing mode. This feature instantly restores the dashboard to its original configuration, providing a clean slate for new customizations.
